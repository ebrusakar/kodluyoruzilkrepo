Patika Aracılığıyla Alınan Eğitimler

Frontend Web Patikası

Büyülü web dünyasına hoş geldin! Eskiler derler ki web dünyasının sihiri tamamen frontend'de saklıdır. Bu atasözünü kim söylediyse kesinlikle doğru söylemiş, web dünyasında biraz sihir yapmak istiyorsan doğru patikaya başlamak üzeresin! Bu büyülü dünyada yolunu bulabilmek için sana birçok teknoloji gösterip sonunda da muhteşem React ile tanıştıracağız. React da seninle tanışmak için oldukça heyecanlı! Eğer sen de heyecanlıysan: You shall pass!

Java ile Backend Web Development Patikası

Büyülü backend dünyasına hoş geldin! Bir söz vardır: Dünyanın yükünü backend'ciler çekermiş. Ama kimse onları tanımazmış. Backend'ciler bizim için gizemli yöntemler kullanır, dünyalarını korurlarmış. Bu çok doğru! Backend alanına başlamak ve bunu da büyülü dünyanın en kadim dillerinden olan Java ile sürdürmek istiyorsan, çok doğru bir patikadasın! Şimdiden uyaralım: Bu dünya karanlık ekranlar ve çokça API içerir!

.Net Core Patikası

.Net Core Microsoft tarafından açık kaynak kodlu (open-source) olarak geliştirilmesi devam eden,cross platform olarak çalışan, esnek ve modern geliştirme platformudur. Gelişen teknolojiyle beraber birçok yazılım şirketi .NET bilen uzmanları tercih etmektedir. Pekii biz bu yolda nasıl ilerleyeceğiz? Aklında bu sorunun canlandığını hissediyorum! Bu yolculuk için sana harika bir yol haritası çıkardık 😄 Hadi vakit kaybetmeden patikaya başlayalım!
